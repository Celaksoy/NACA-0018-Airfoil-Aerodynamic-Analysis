# Airfoil Analysis of NACA 0018 (CFD, XFOIL & Vortex Panel Method)

## Project Description

This project focuses on the aerodynamic analysis of the NACA 0018 airfoil using multiple computational approaches.

The airfoil is assigned based on student ID, and in this study, NACA 0018 is analyzed using:

* Computational Fluid Dynamics (CFD)
* XFOIL
* A custom-developed Vortex Panel Method code

The aim is to compare different analysis methods and evaluate their accuracy and performance.

##  Methodology

###  CFD Analysis

* Software: ANSYS Fluent
* Mesh generation and boundary condition setup
* Aerodynamic coefficient extraction (Lift, Drag)

### XFOIL Analysis

* Airfoil performance analysis under varying conditions
* Lift and drag coefficient estimation
* Fast, low-cost aerodynamic evaluation

### Vortex Panel Method (Custom Code)

* Implementation of clustered vortex panel method
* Numerical solution of potential flow
* Pressure distribution and lift calculation

## Tools & Technologies

* ANSYS Fluent
* XFOIL
* MATLAB 

## How to Run

1. Read report
2. Execute the vortex panel method code
3. Open CFD results in ANSYS Fluent
4. Compare aerodynamic results across methods

## Key Insights

* CFD provides high accuracy but requires significant computational cost
* XFOIL offers fast and reasonably accurate results
* Vortex panel method demonstrates fundamental aerodynamic behavior with simplified assumptions
